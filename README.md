# Análise Exploratória de Dados - Censo Escolar 2025

## 📌 Descrição do Projeto

Este trabalho tem como objetivo realizar uma análise exploratória de dados (EDA) utilizando os microdados do Censo Escolar 2025, com foco na tabela de escolas (`Tabela_Escola_2025.csv`).

A análise busca compreender características estruturais e administrativas das escolas brasileiras, identificando padrões e possíveis desigualdades.

---

## 🎯 Objetivos

- Explorar os dados de escolas do Brasil
- Aplicar estatística descritiva
- Identar padrões e relações entre variáveis
- Gerar visualizações para melhor interpretação dos dados

---

## 📊 Etapas Realizadas

### 1. Carregamento dos dados

- Importação do dataset e do dicionário de dados
- Inspeção inicial das colunas e tipos de dados

### 2. Tratamento de dados

- Verificação de valores ausentes
- Seleção de variáveis relevantes
- Mapeamento de variáveis categóricas (ex: situação de funcionamento)

### 3. Análise estatística

Foram calculadas as seguintes métricas:

- Média, mediana e moda
- Valores mínimos e máximos
- Desvio padrão e variância
- Quartis e intervalo interquartil (IQR)
- Contagem de valores ausentes
- Tipos de dados das colunas

### 4. Visualização de dados

Foram gerados gráficos utilizando matplotlib:

- Histogramas (distribuição dos dados)
- Boxplots (detecção de outliers)
- Gráficos de dispersão (relação entre variáveis)

---

## 📈 Principais Insights

- Diferenças relevantes entre escolas urbanas e rurais
- Relação entre infraestrutura e porte da escola
- Desigualdade no acesso a recursos como internet e biblioteca

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib

---

## 📂 Estrutura

- `notebook.ipynb` → análise completa
- `Tabela_Escola_2025.csv` → base de dados
- `dicionário_dados.xlsx` → descrição das variáveis

---

## ✅ Conclusão

A análise demonstrou como técnicas de exploração de dados podem revelar padrões importantes no sistema educacional brasileiro, auxiliando na compreensão de desafios estruturais e na tomada de decisões.
